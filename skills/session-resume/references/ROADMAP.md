# Roadmap - session-resume

## Potential Future Enhancements

**Archive Features** (if users request):
- Archive cleanup (delete old archives)
- Archive statistics (session frequency)
- Archive search by content
- Archive comparison (diff between sessions)

**Branch Awareness**:
- Different resumes per git branch
- Detect branch switches
- Auto-archive on branch change

**Multi-Project**:
- Resume across related projects
- Project dependency tracking
- Workspace-level resume views

## Non-Goals

- Auto-invocation (explicit triggering only)
- Resume editing from within skill
- Integration with external tools
